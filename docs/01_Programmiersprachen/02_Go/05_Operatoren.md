## Rechenzeichen

* Addiert = +
* Subtrahiert = -
* Multiplizier = *
* Dividiert = /
* Modul-Operator = %
* Inkrement-Operato = ++
* Dekrementieren = ```--```



## Vergleichsoperator

* gleich = ==
* nicht gleich= !=
* größer als = >
* kleiner als = <
* größer oder gleich = >=
* kleiner oder gleich = <=



## Logische Operatoren

* und = &&
* oder = ||
* nicht = !



## Zuweisungsoperatoren

* zuweisen = =
* addieren zuweisen = +=
* Subtrahiert zuweisen = -=
* Multiplizier zuweisen = *=
* Dividiert zuweisen = /=
* Modul-Operato zuweisen = %=
